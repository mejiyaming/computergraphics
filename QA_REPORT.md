## 2. 세부 검토 의견 및 보완 가이드

### ① 과제 요구사항 충족 여부
* **게임 실행 상태 (Pass)**: 로컬 호스팅 및 빌드 테스트 완료. `EndingScene` 카메라 타겟 미지정 버그 등 모든 런타임 오류가 수정되어 정상 작동하며, 3개 젬 획득 시 최종 포탈이 활성화되고 엔딩 날개 비행 후 스크롤 크레딧까지 완벽하게 완결됩니다.
* **증빙용 스크린샷 누락 (심각)**: 12개 요구 스크린샷 중 오프닝 시작 화면(`start_screen.jpg`)만 제공되었고, **나머지 11개의 인게임 플레이/기술 증빙 스크린샷이 부재**합니다. 교수가 리포트를 뷰어로 읽을 때 엑박으로 노출되며, 캡처가 없을 시 0점 처리하겠다는 조건에 따라 큰 감점 요소가 됩니다.

### ② Git 저장소 구조 적절성
* **초기 상태 (부적절)**: 로컬 저장소 자체가 초기화되지 않았고 `.gitignore`가 없어 그대로 푸시할 경우 대용량 `node_modules`와 `dist` 빌드 폴더가 모두 동기화되는 문제가 있었습니다.
* **개선 조치**: 로컬 저장소(`git init`)와 원격 저장소(`https://github.com/mejiyaming/computergraphics.git`)를 연결하고, 빌드 부산물 및 임시 파일들을 걸러내는 `.gitignore` 파일을 루트에 정상 구축해 두었습니다.

### ③ GitHub Pages 또는 웹 배포 상태
* **배포 여부 (미완료)**: 로컬 코드만 존재할 뿐, 원격 저장소로 `git push`가 이루어지지 않아 깃허브 상에 아무 자료도 없는 상태입니다. 또한 라이브 배포 주소(GitHub Pages)가 설정되어 있지 않아 교수가 웹에서 직접 테스트할 수 없습니다.

### ④ Markdown 리포트 구성 적절성
* 리포트의 체계성과 목차 배치는 훌륭합니다. 다만, 수식이 텍스트로 길게 풀어져 있어 학술성이 다소 떨어져 보입니다. 교수가 리포트를 평가할 때 수식이 깔끔하게 LaTeX로 표현되어 있으면 고급스러운 느낌을 주어 리포트 40점 만점을 확보하는 데 유리합니다.

## 3. 리포트 내 GI(Global Illumination) 기술 설명 보완 제안
교수의 학술적 평가 눈높이에 맞춰, `REPORT.md` 내에 기재된 DDGI 및 SurfelGI 이론 수식을 LaTeX 표기법으로 업그레이드할 것을 제안합니다.

### 3.1 DDGI (Dynamic Diffuse Global Illumination) 프로브 적산식 고도화
기존의 가독성 떨어지는 텍스트 수식 대신, 아래의 **조도 적산 렌더링 방정식(Render Equation)**을 LaTeX 표현식으로 삽입하면 논문 수준의 리포트로 평가받을 수 있습니다.
\[E(\vec{p}, \vec{n}) = \int_{\Omega} L_i(\vec{p}, \vec{\omega}_i) (\vec{n} \cdot \vec{\omega}_i) d\vec{\omega}_i \approx \sum_{k=1}^{N} w_k(\vec{p}) \cdot I_k\]
* \(\vec{p}\): 표면상의 특정 충돌 좌표
* \(\vec{n}\): 표면의 법선 벡터(Normal)
* \(I_k\): 각 Virtual Irradiance Probe(가상 조도 탐침)가 표면에 가하는 광선 감쇄 강도

### 3.2 SurfelGI (Surfel-based Global Illumination) 적산식 도입
수정이 배치된 제단이나 벽면 테두리의 고휘도 에미시브 서펠(Surfel Emitter) 광역 효과를 설명하기 위해, 서펠 간의 폼 팩터(Form Factor) 및 조도 기여 공식을 리포트에 보강합니다.
\[L_{indirect}(x) = \sum_{s \in \text{Surfels}} B_s \cdot F(s, x) \cdot V(s, x)\]
* \(B_s\): 서펠 표면의 발광도(Radiosity / Emissive intensity)
* \(F(s, x)\): 서펠과 표면 점 \(x\) 사이의 기하학적 형태 인자(Form Factor)
* \(V(s, x)\): 두 점 사이의 가시성(Visibility / Ambient Occlusion 효과)
